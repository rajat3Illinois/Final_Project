Skyscraper


Overview

The dataset published by named as skyscrapers.csv contains collected statistics about the tallest buildings all over the United States, including the materials used to develop them, the purpose of the buildings, and more.

Below is the plot of the Location City count. One can select the material type that changes the visualization. Selection Tag is used to dynamically the change the plot based on the material type.

The geospatial plot is depicted using the longitude and latitude data provided in the skyscrapers.csv is used in the visualization. Toolkit is added, that displays the name of the building when hovering.

There are two visualizations plotted below, first is Dynamic Corner Radius Plotting the material type and second is depicting the Binned Completed year Count of Buildings.
